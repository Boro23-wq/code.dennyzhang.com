A boomerang is a set of 3 points that are all distinct and not in a straight line.

Given a list of three points in the plane, return whether these points are a boomerang.

Example 1:
#+BEGIN_EXAMPLE
Input: [[1,1],[2,3],[3,2]]
Output: true
#+END_EXAMPLE

Example 2:
#+BEGIN_EXAMPLE
Input: [[1,1],[2,2],[3,3]]
Output: false
#+END_EXAMPLE
 
Note:

1. points.length == 3
- points[i].length == 2
- 0 <= points[i][j] <= 100
