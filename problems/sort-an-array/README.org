Given an array of integers nums, sort the array in ascending order.

Example 1:
#+BEGIN_EXAMPLE
Input: [5,2,3,1]
Output: [1,2,3,5]
#+END_EXAMPLE

Example 2:
#+BEGIN_EXAMPLE
Input: [5,1,1,2,0,0]
Output: [0,0,1,1,2,5]
#+END_EXAMPLE
 
Note:

1. 1 <= A.length <= 10000
- -50000 <= A[i] <= 50000

- Solution:

#+BEGIN_SRC go
// Blog link: https://code.dennyzhang.com/sort-an-array
// Basic Ideas: quicksort
// Complexity: Time O(1), Space (1)
import "sort"
func sortArray(nums []int) []int {
    sort.Ints(nums)
    return nums
}
#+END_SRC
