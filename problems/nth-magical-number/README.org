A positive integer is magical if it is divisible by either A or B.

Return the N-th magical number.  Since the answer may be very large, return it modulo 10^9 + 7.

Example 1:
#+BEGIN_EXAMPLE
Input: N = 1, A = 2, B = 3
Output: 2
#+END_EXAMPLE

Example 2:
#+BEGIN_EXAMPLE
Input: N = 4, A = 2, B = 3
Output: 6
#+END_EXAMPLE

Example 3:
#+BEGIN_EXAMPLE
Input: N = 5, A = 2, B = 4
Output: 10
#+END_EXAMPLE

Example 4:
#+BEGIN_EXAMPLE
Input: N = 3, A = 6, B = 4
Output: 8
#+END_EXAMPLE
 
Note:

1. 1 <= N <= 10^9
- 2 <= A <= 40000
- 2 <= B <= 40000
