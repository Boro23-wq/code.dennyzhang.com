Sum of Square Numbers
---------------------------------------------------------------------
Given a non-negative integer c, your task is to decide whether there're two integers a and b such that a*a + b*b = c.
#+BEGIN_EXAMPLE
Example 1:
Input: 5
Output: True
Explanation: 1 * 1 + 2 * 2 = 5
#+END_EXAMPLE

#+BEGIN_EXAMPLE
Example 2:
Input: 3
Output: False
#+END_EXAMPLE
